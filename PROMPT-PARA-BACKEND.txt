PROBLEMA CRÍTICO: ENDPOINTS DE OBRAS DESHABILITADOS - SISTEMA NO FUNCIONAL

CONTEXTO:
Tengo un frontend React que gestiona obras de construcción. La base de datos tiene 9 obras registradas (obras independientes, trabajos extra y obras principales), pero la interfaz muestra 0 obras porque los endpoints principales están deshabilitados con el error "Método temporalmente deshabilitado - pendiente arreglo MapStruct".

ERRORES ACTUALES EN CONSOLA:
- GET /api/obras/todas?idEmpresa=1 → 400 Bad Request
- GET /api/obras/empresa/1?empresaId=1 → 400 Bad Request
- GET /api/obras/activas?empresaId=1 → 400 Bad Request
- GET /api/obras/estado/{estado}?idEmpresa=1 → 400 Bad Request

Todos devuelven: "Método temporalmente deshabilitado - pendiente arreglo MapStruct"

ENDPOINT ÚNICO DISPONIBLE:
Solo funciona GET /api/obras/borradores?empresaId={id} pero este ÚNICAMENTE devuelve obras en estado BORRADOR, excluyendo todas las obras en estados: A_ENVIAR, APROBADO, EN_PROGRESO, COMPLETADO, SUSPENDIDA, CANCELADO.

PROBLEMA PRINCIPAL:
Las 9 obras en base de datos no se pueden visualizar porque:
1. Los endpoints generales están deshabilitados
2. El único endpoint disponible solo muestra estado BORRADOR
3. No hay forma de listar TODAS las obras de una empresa

SOLUCIÓN REQUERIDA (PERSISTENTE Y ROBUSTA):
Necesito que implementes UN endpoint que devuelva TODAS las obras de una empresa, sin importar su estado. Este debe ser el endpoint principal para consulta de obras.

IMPLEMENTACIÓN BACKEND REQUERIDA:

Controller (ObraController.java):
@GetMapping("/api/obras")
public ResponseEntity<List<ObraResponseDTO>> listarTodasObras(@RequestParam Long empresaId) {
    List<ObraResponseDTO> obras = obraService.listarTodasObrasPorEmpresa(empresaId);
    return ResponseEntity.ok(obras);
}

Service (ObraService.java):
public List<ObraResponseDTO> listarTodasObrasPorEmpresa(Long empresaId) {
    List<Obra> obras = obraRepository.findByEmpresaIdOrderByFechaCreacionDesc(empresaId);
    return obras.stream()
        .map(obraMapper::toResponseDTO)
        .collect(Collectors.toList());
}

Repository (ObraRepository.java):
@Query("SELECT o FROM Obra o WHERE o.empresaId = :empresaId ORDER BY o.fechaCreacion DESC")
List<Obra> findByEmpresaIdOrderByFechaCreacionDesc(@Param("empresaId") Long empresaId);
